# AITL Paper: AI In The Loop
paperlink -https://zenodo.org/records/19551173 
This repository contains the LaTeX source, documentation, and experimental Proof-of-Concept code for the research paper:

**"AI In The Loop: The Evolution from Human-in-the-Loop Evaluation"**

## Overview

AI In The Loop (AITL) is a paradigm where AI systems autonomously generate, evaluate, and improve without human intervention in operational workflows. AITL extends the RLAIF principle—replacing human feedback with AI feedback—from training to the full AI system lifecycle.

## Repository Structure

- `paper/`: LaTeX source files, compiled PDF, and figures.
- `experiments/`: Proof-of-Concept code demonstrating the AITL loop (Blind NAS Tuner).
- `docs/`: Standalone markdown guides for AITL principles and definitions.

We identify and formalize AI In The Loop (AITL), a paradigm where AI systems autonomously generate, evaluate, and improve without human intervention in operational workflows.
